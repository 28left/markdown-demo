An **arithmetic progression (AP)** of length $k$ is a sequence $a, a+m, a+2m, \ldots, a+(k-1)m$.

> **THM:** For any $r, k \geq 1$, there exists $W(r,k)$ such that any $r$-coloring of $\{1, \ldots, W(k,r)\}$ has a monochromatic $k$-AP.

Gowers showed that

$$
W(r,k) \leq 2^{2^{r^{2^{2^{k+9}}}}
$$
